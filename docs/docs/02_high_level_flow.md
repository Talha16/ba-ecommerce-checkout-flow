# High-Level Checkout Flow

## Overview
This document outlines the primary steps a customer follows when completing a purchase on NovaCart.

## Checkout Steps
1. Review cart items
2. Proceed to checkout
3. Choose sign-in or guest checkout
4. Enter shipping address
5. Select shipping method
6. Enter payment details
7. Review order summary
8. Place order
9. Receive order confirmation

## Common Exception Scenarios
- Invalid or incomplete shipping address
- No shipping options available for selected address
- Payment authorization failure
- Session timeout during checkout
